# BulkBins Business Intelligence System

## Project Overview
BulkBins is a premium retail management platform designed to provide institutional-grade business intelligence and automated financial tracking.

## Module Spotlight: Profit & Inventory Analysis
The **Profit & Inventory Analysis Module** automatically calculates profits, tracks inventory for COGS computation, and generates low-stock alerts. It enables AI-driven decision-making for inventory optimization and profitability improvement.

### Key Capabilities:
- **Automated Financials**: Real-time Gross and Net Profit calculations with direct COGS integration.
- **Inventory Lifecycle Management**: Automated stock synchronization across sales, modifications, and deletions.
- **AI Intelligence Hub**: Predictive demand forecasting, actionable reorder recommendations, and Profit Star identification.
- **Dynamic Alerts**: Multi-layer notification system for critical stock levels.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide Icons, Chart.js.
- **Backend**: Flask (Python), SQLAlchemy, SQLite, Scikit-learn (AI Services).
